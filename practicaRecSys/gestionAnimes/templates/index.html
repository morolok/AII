Bienvenido al Sistema de Recomendacion de Animes Dataset!!!

Construir el modelo de datos en Django que almacene la información siguiente:

a) Anime: Animeid, Título, Géneros, Formato de emisión (TV, movie,…), Número de episodios.
b) Puntuación: IdUsario, Animeid, Puntuación (1-10) Poblar la base de datos extrayendo los datos del Anime Dataset que acompaña este enunciado.

Se pide construir una web con un menú principal, que permita las siguientes opciones:

a) CARGAR LA BASE DE DATOS. Muestre un formulario de confirmación. Si se acepta, borrar la base de datos y volverla a crear con los datos de la web. Después MOSTRAR INFORMACIÓN DE LOS REGISTROS ALMACENADOS EN CADA TABLA que hayáis definido en el modelo.
b) CARGAR RECSYS. Que cargue el Sistema de Recomendación.
c) PUNTUACIONES DE UN USUARIO. Dado un usuario (IdUsuario), muestre los animes que ha puntuado (Animeid, Título y Puntuación que le ha dado).
d) ANIMES POR GÉNERO. Muestre un cuadro combinado con los géneros que hay. Una vez que se seleccione uno muestre todos los animes de ese género (Título, Géneros, Formato de emisión y Número de episodios)
e) USUARIOS PARECIDOS. Dado un usuario (IdUsuario), presente los tres usuarios que más se parecen (IdUsuario y Coeficiente de similaridad).
f) RECOMENDAR ANIMES. Dado un usuario (IdUsuario), le recomiende cinco animes que no haya puntuado (Animeid, título y puntuación propuesta por el sistema de recomendación), usando un sistema de recomendación de tipo filtrado colaborativo basado en ítems.
